# OzCrypto Company Website

## Cara Membuka File HTML

### Online (GitHub Pages)
1. Kunjungi: https://kurobenedict-hash.github.io/OzCryptos-websites/
2. Halaman akan menampilkan OzCrypto dengan animasi LED neon

### Local/Offline
1. Download atau clone repository ini
2. Buka file `Index.html` di browser Anda
3. Klik tombol "Baca" untuk melihat lampiran PDF

## File Penting
- `Index.html` - Halaman utama dengan design LED neon
- `lampiran.html` - Halaman PDF viewer
- `lampiran.pdf` - File company profile
- `Company Profile OZCRYPTO.pdf` - Document asli
